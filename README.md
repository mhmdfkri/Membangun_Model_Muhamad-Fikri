# Telco Customer Churn Prediction

## Deskripsi
Project machine learning untuk memprediksi churn pelanggan telekomunikasi menggunakan Random Forest Classifier.

## Dataset
Telco Customer Churn Dataset

## Struktur Folder

data/
preprocessing/
training/
notebook/

## Menjalankan Preprocessing

```bash
cd preprocessing
python automate_Fikri.py
```

## Menjalankan Training

```bash
cd training
python modelling.py
```
